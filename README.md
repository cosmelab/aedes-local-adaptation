# Population Genetics Analysis Project

## Directory Structure

- **data/**: Input data files
  - raw/: Original data files
  - processed/: Cleaned and filtered data
  - references/: Reference genomes and annotations
  - metadata/: Sample and population information

- **results/**: Analysis outputs
  - interim/: Intermediate results
  - organized/: Final results
  - analysis/: General analyses
  - populations/: Population structure
  - local_adaptation/: Selection and adaptation analyses
  - probes/: Marker analyses
  - segregation/: Segregation analyses
  - global_brazil/: Geographic analyses
  - ba3/: BayesAss3 outputs
  - admixture/: ADMIXTURE outputs

- **scripts/**: Analysis scripts
  - download/: Data retrieval scripts
  - analysis/: Processing pipelines
  - visualization/: Plotting scripts

- **configs/**: Configuration files
- **containers/**: Container definitions
- **logs/**: Process logs
- **output/**: Additional outputs
